   Format.ringFill(oneIn: match.myPick.exactChanceOneIn), color: Format.tierRingColor(match.myPick.tier),
   size: 140) { Text(match.myPick.score).font(.display) }`; below, a `SoftPill` tier and either a

   a row with a `checkmark` in `Theme.posText` + the reason (`.bodyX`). Stagger the bullets.
3. **The matchup** (`FrostCard`): `Eyebrow("The matchup")`; `StrengthBar(...)`; a line built from
   `Format.strengthWords(match.favoriteStrength)` + `match.favorite` (+ if `match.upset.isUpset`, a
   `exclamationmark.triangle` in `Theme.negText` + `match.upset.text`).
4. **The room** (`FrostCard`): `Eyebrow("The room")`; an INLINE dots row — `HStack` of
   `match.fieldExposure.friendsTotal` circles, the first `onSamePick` filled `Theme.iridLilac`, the
   rest `Theme.track`; a sentence: if `onSamePick >= friendsTotal/2`
   `"\(onSamePick) of \(friendsTotal) friends are here too — safe, low separation."` else
   `"Only \(onSamePick) of \(friendsTotal) friends land here — that's your separation."`; then the
   leverage + vsLeader `SoftPill`s (via `Format.leverageChip` / `Format.leaderChip(... match? leader name)`);
   and if `leaderPick` non-nil, `"Leader's likely tip: \(leaderPick)"`.

   `match.alternates`, a row: `Text(alt.score)` + a thin iridescent bar whose width =
   `alt.expectedPoints / maxAltEP` (RELATIVE length only, **no number shown**) + `alt.note`; tag the
   one equal to `match.myPick.score` with a "Your pick" `SoftPill`.
6. **News** (if any): `Eyebrow("News")`; each `match.news` item with a `newspaper` icon (`.bodyX`, dim).
